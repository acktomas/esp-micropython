# 硬件参数解析

## 1. OLED液晶屏模块参数
- **尺寸**：0.96寸
- **分辨率**：128*64像素
- **通信方式**：I2C (Inter-Integrated Circuit)，也称为IIC
- **驱动芯片**：SSD1315

**作用**：用于实时显示电机当前角度、目标角度、转速等参数，方便调试和监控。

## 2. 带霍尔编码器的直流减速电机参数
- **额定电压**：12V
- **空载转速**：333转/分钟
- **额定转速**：250转/分钟
- **减速比**：30:1（电机轴每转30圈，输出轴转1圈）
- **编码器PPR**：11（根据用户说明，电机转动一圈产生11个脉冲）

### 关键计算
- **电机输出轴每转的脉冲数**：11脉冲/转（用户明确指出）
- **角度分辨率**：360度 / 11脉冲 ≈ 32.73度/脉冲（不考虑减速比）
- **考虑减速比后**：
  - 电机轴转30圈，输出轴转1圈
  - 电机轴每转的角度：360度 / 30 = 12度
  - 输出轴每转的总脉冲数：11脉冲/电机轴转 × 30电机轴转/输出轴转 = 330脉冲/输出轴转
  - **综合角度分辨率**：360度 / 330脉冲 ≈ 1.09度/脉冲（输出轴）

## 3. 主控板：ESP32-WROOM
- Wi-Fi和蓝牙双模微控制器
- 丰富的GPIO引脚，支持PWM输出和外部中断
- 支持MicroPython编程

## 4. 电机驱动模块：L298N
- 双H桥直流电机驱动芯片
- 控制电机的正反转和转速（通过PWM信号）
- 最大电流：2A（每通道）

## 控制思路
1. **闭环控制**：利用编码器反馈实现精确角度控制
2. **脉冲计数**：通过计数编码器脉冲确定当前角度
3. **PWM调速**：通过调节PWM信号占空比控制电机转速
4. **方向控制**：通过L298N的IN引脚控制电机正反转
